# Card Game

**Online multiplayer card game**

## Setup

```
npm install -g @vercel/ncc

cd server
npm install

cd client
npm install
```

### Development

```
cd server
npm run watch
```

```
cd client
npm run watch
```

### Production
**Build**

```
cd server
npm run build

cd client
npm run build
```

**Start**

`cd server && npm run start`
